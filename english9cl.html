Present Simple vs Present Continuous
Present Simple: використовується для регулярних дій, фактів та звичок. Формула: Subject + Verb(s). Present Continuous: використовується для дій, які відбуваються зараз. Формула: Subject + am/is/are + Verb(-ing).
Past Simple vs Past Continuous
Past Simple: для завершених дій у минулому. Формула: Subject + Verb(-ed) або 2-га форма неправильного дієслова. Past Continuous: для дій, що тривали в певний момент у минулому. Формула: Subject + was/were + Verb(-ing).
Future Tenses
Future Simple: використовується для прогнозів, спонтанних рішень. Формула: Subject + will + Verb. Going to: для запланованих дій або намірів. Формула: Subject + am/is/are going to + Verb.
Conditional Sentences
Zero Conditional: для фактів та істин. Формула: If + Present Simple, Present Simple. First Conditional: для реальних ситуацій. Формула: If + Present Simple, will + Verb. Second Conditional: для уявних ситуацій. Формула: If + Past Simple, would + Verb.
Passive Voice
Використовується, коли дія важливіша за виконавця. Формула: Subject + be + Past Participle (V3). Час змінюється за допомогою форми "be".
Reported Speech
Передача чиїхось слів. Зміна часу відбувається відповідно до правил: Present → Past, Past → Past Perfect. Заміна займенників та обставин часу/місця.
Modal Verbs
Використовуються для вираження можливості, дозволу, обов’язку тощо. Основні модальні дієслова: can, could, may, might, must, have to, should, would.
Articles (a, an, the)
A/An: для невизначених предметів (a перед приголосними, an перед голосними звуками). The: для конкретних предметів або тих, які вже згадані.
Word Formation
Використання префіксів (un-, im-, dis-) та суфіксів (-ful, -less, -ly) для створення нових слів. Наприклад, happy → happiness.
Writing Emails and Letters
Основні частини листа: привітання (Dear ...), вступ, основний текст, завершення (Best regards). Формальність залежить від адресата.
